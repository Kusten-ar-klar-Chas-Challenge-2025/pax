## Rooms:
- Hämta alla rum
- Bokningsstatus 
- Id
- Data bokat rum: 
1. Tid/längd bokat (från kl 00.00 - till kl 00.00)
2. Tid kvar på bokning
3. Luftkvalité - bra/underkänd (notis?)

- Data obokat rum 
1. antal platser
2. storlek i m2
3. webcam
4. skärm
5. whiteboard
6. projektor

Hur visa data?

- Överblick/lista
- Karta
- Våning
- Ritning karta 2D (ev 3D-karta)

- Boka - “paxa”

- Avbokas vid inaktivitet i X antal min

- Alt: ”Är bokat men tomt”

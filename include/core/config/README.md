# Configuration files

## arduino_secrets.h
Do not track this file in the repository.
Example:
- #define SSID "your_ssid"
- #define PASSWORD "your_password"
- #define SERVER_HOST "your_server_host"
- #define SERVER_PORT 80U

## pin_config.h
Pin assignments for sensors and similar according to wiring diagram.
If any pins are changed, update this file.
